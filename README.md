# Сравнение роутеров под OpenWrt

Небольшая таблица сравнения роутеров, которые я рассматриваю для OpenWrt.

Онлайн-версия:
https://altiqo.github.io/openwrt-router-compare/

## Что внутри

- таблица собирается из `data.csv`
- примечания берутся из `notes.csv`
- страница полностью статическая
- проект можно публиковать через GitHub Pages

## Модели в таблице

- Cudy TR3000 / TR30
- Cudy WR3000S v1
- Cudy WR3000E v1
- Cudy TR1200
- Cudy WR1300 v3
- Netis NX31
- Netis N6
- Netis NX32U
- Xiaomi AX3000T

## Как редактировать

### Изменить список роутеров
Открой `data.csv` и обнови строки.

### Изменить текст на странице
Открой `index.html`.

### Изменить внешний вид
Открой `style.css`.

## Публикация

1. Залей файлы в свой репозиторий GitHub
2. Открой `Settings` → `Pages`
3. Включи публикацию из ветки `main`
4. Выбери папку `/(root)`

После этого сайт будет доступен по адресу:

https://ТВОЙ_ЛОГИН.github.io/ТВОЙ_РЕПО/
