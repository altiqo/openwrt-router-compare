# Сравнение роутеров под OpenWrt

Небольшая таблица сравнения роутеров, которые я рассматриваю для OpenWrt.

Онлайн-версия:
https://altiqo.github.io/openwrt-router-compare/

## Что внутри

- таблица собирается из `data.csv`
- примечания берутся из `notes.csv`
- страница полностью статическая
- проект можно публиковать через GitHub Pages
